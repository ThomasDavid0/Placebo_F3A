Placebo F3A Plane Build Manual

This site contains instructions to build the Placebo F3A Plane.

Photos of the prototype build can be seen in the following links:

  • Fuselage
  • Wing
  • Tail

Here is a list of parts used in the prototype, note these are local suppliers, please submit a PR with alternatives / global suppliers:
  • Wing Tube
  • Tail Tube
  • Landing Gear
  • TailWheel
  • Canopy Latch


DXF files

Balsa parts for the fuselage. cut from 3/32" (or 2.5mm) light balsa sheets. Parts are nested on 48" x 4" (1200mm * 100mm) sheets:
  • Canopy Sheet 1
  • Canopy Sheet 2 and rudder
  • Canopy Sheet 1
  • Turtle Deck 1
  • Belly
  • Turtle Deck 2
  • Box 1
  • Box 2
  • Nose

Balsa parts for the wings and tail. Cut two of each file from 3/32" (or 2.5mm) light balsa sheets. Parts are nested on 48" x 4" (1200mm * 100mm) sheets:
  • Wing 1
  • Wing 2
  • Wing 3
  • Wing 4
  • Wing 5
If you can't access 48" lengths, Roland Poidevin has rearranged the parts onto 1000mm lengths here.

Balsa blocks for shaping the fuselage. Cut from 1/4" (6.5mm?) light balsa sheets:
  • Fuselage Blocks
Plywood parts for the fuselage, wings and tail. Cut from 1/16" (1.5mm) birch plywood:
  • Birch Ply 1
  • Birch Ply 2
  • Birch Ply 3
Control horns. Cut from 1/16" (1.5mm) gfrp sheet:
  • Control Horns
